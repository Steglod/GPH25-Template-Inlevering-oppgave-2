# Guide til webutvikling

## header-elementet

- Logo
- Søkemotor
- Logg inn / registrer
- Navigasjon

## main-elementet

- "Hero" det viktigaster på sida
  - Info om kva sida er
  - Spesielle tilbud
  - Nytt som skjer
- Tilegg seksjon med meir info
  - Kort med informasjon
  - Dypere navigasjon
  - Reklame for siden/produkt på siden

## footer-element

- Logo
- Navigasjon
- Personverserklæring / TOS
- Sosiale Media
- Kreditasjon
